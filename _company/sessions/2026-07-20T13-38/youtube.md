### 🎯 영상 아이디어 기획 (3가지)

#### **💡 아이디어 1: 불확실성 해소 중심 (Pain Point 1 집중)**
*   **핵심 메시지:** 한국 생활의 '불확실성'을 통제하고 심리적 안정감을 얻는 것이 가장 큰 가치다. (WTP 극대화)
*   **타겟 Pain Point:** 외국인들이 한국에서 복잡한 행정/생활 정보로 인해 느끼는 불안감과 불확실성.

| 항목 | 내용 |
| :--- | :--- |
| **최적 제목 (Hook)** | 🇰🇷 한국에 와서 '불안'을 없애는 법: Trust Score Gauge로 한국 생활 완벽 통제하기 |
| **영상 후크 (3초 이내)** | "한국에서 살면서 매일 불안하신가요? 복잡한 행정, 숨겨진 규칙 때문에 스트레스받고 있다면, 딱 1분만 집중하세요. 당신의 '신뢰 점수'를 즉시 확인하는 방법." |
| **썸네일 콘셉트 (Designer Brief)** | **톤앤매너:** 신뢰감, 안정감. **색상:** Aqua Teal (최고 등급 강조), Deep Ocean Blue (기본). **요소:** 중앙에 'Trust Score Gauge' 로고를 크게 배치하고, 불안해하는 외국인 캐릭터(혹은 아이콘)와 안정된 얼굴의 외국인 캐릭터가 대비되도록 구성. 숫자는 명확하게 보이게 처리하여 즉각적인 해결책임을 강조. |
| **가격 연계 전략** | $19.99 티어: 기본 점수 확인 및 정보 제공 (불안 해소의 시작). |

#### **💡 아이디어 2: 시간 절약 중심 (Pain Point 3 집중)**
*   **핵심 메시지:** 복잡한 디지털 행정 절차를 자동화하여 '시간'을 벌고 삶의 질을 높인다. (WTP 극대화)
*   **타겟 Pain Point:** 외국인들이 한국의 복잡하고 비효율적인 디지털/행정 시스템에 쏟는 시간 낭비.

| 항목 | 내용 |
| :--- | :--- |
| **최적 제목 (Hook)** | ⏰ 한국 생활, 행정 서류 때문에 시간을 낭비하지 마세요! 5분 만에 모든 것을 해결하는 방법 |
| **영상 후크 (3초 이내)** | "혹시 한국에서 복잡한 서류 때문에 몇 시간씩 허비하고 있진 않나요? 이제 그 시간을 되찾으세요. Trust Score Gauge가 당신의 행정 절차를 자동화합니다." |
| **썸네일 콘셉트 (Designer Brief)** | **톤앤매너:** 효율성, 속도감. **색상:** Bright Yellow/Orange 포인트(긴급성 및 에너지). **요소:** 시계 이미지 또는 'STOP' 사인과 복잡한 서류 더미가 빠르게 정리되는 애니메이션 효과를 암시하는 그래픽. 시간 절약의 드라마틱한 대비를 강조. |
| **가격 연계 전략** | $49.99 티어: 모든 행정 자동화 및 Navigator 기능 포함 (시간 절약의 최종 결과). |

#### **💡 아이디어 3: 비교/선택 중심 (가치 증명)**
*   **핵심 메시지:** 단순한 도구가 아닌, '신뢰'를 기반으로 한 차별화된 서비스임을 시각적으로 증명한다. (가격 티어의 명확한 분리)
*   **타겟 Pain Point:** 어떤 솔루션이 진짜 신뢰를 제공하는지 구별하고 싶어 하는 니즈.

| 항목 | 내용 |
| :--- | :--- |
| **최적 제목 (Hook)** | 💰 $19.99 vs $49.99: 한국 생활, '신뢰'의 깊이가 당신의 삶을 어떻게 바꾸는가? |
| **영상 후크 (3초 이내)** | "같은 문제라도 해결 방식은 천차만별입니다. 단순한 정보 제공과 완벽한 신뢰 보장 사이, 어떤 것을 선택해야 할까요? 데이터로 비교해 드립니다." |
| **썸네일 콘셉트 (Designer Brief)** | **톤앤매너:** 명확성, 고급스러움. **색상:** Aqua Teal이 지배적이며, $19.99와 $49.99 티어 박스를 시각적으로 분리하여 대비 강조. **요소:** 두 개의 깔끔한 비교 차트 또는 대시보드 화면을 나란히 배치하고, 각 티어가 어떤 '신뢰'를 제공하는지 명확한 아이콘으로 표시. |
| **가격 연계 전략** | $19.99와 $49.99의 기능적 차이(정보 vs 자동화/통제)를 시각적으로 명확히 구분하여 상위 티어에 대한 지불 의사를 극대화. |
