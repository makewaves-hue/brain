## 1. Pro Tier 핵심 기능 정의 및 데이터 필드 설계

Pro Tier의 핵심은 **'수동적인 정보 소비'에서 '능동적인 맞춤형 실행 계획 수립'으로 전환**시키는 것입니다. 이를 위해 필요한 데이터 필드와 사용자 플로우를 다음과 같이 설계합니다.

### A. 필수 데이터 필드 (System/User Input)

이 데이터는 AI가 고품질의 로드맵과 답변을 생성하기 위한 근거 자료가 됩니다.

| 데이터 영역 | 구체적인 필드 명 | 설명 및 목적 |
| :--- | :--- | :--- |
| **초기 진단** | 목표 (Goal) | 사용자가 달성하고자 하는 최종 목표 (예: 3개월 내 취업 준비, 6개월 내 한국 생활 적응 등). |
| | 현재 상태 (Current Status) | 사용자의 현재 지식 수준, 경험, 자원(재정, 시간 등)에 대한 정량적 입력. |
| **환경 설정** | 지역/상황 변수 | 거주 도시, 예산 범위, 선호하는 생활 방식 (예: 배낭여행 vs 정착), 문화적 민감도. |
| | 제약 조건 (Constraints) | 시간 제약(하루에 할애할 수 있는 시간), 예산 상한선, 물리적 제약 사항 등. |
| **구체화** | 선호 학습 스타일 | 텍스트 기반 선호, 시각 자료 선호, 실습 중심 선호 등 AI의 출력 방식을 조절. |
| | 선행 경험 입력 | 이미 알고 있는 정보나 시도했던 활동을 입력하여 중복 학습 방지. |

### B. 사용자 플로우 설계 (User Flow)

사용자가 Pro Tier 기능을 경험하는 과정은 **'진단 → 계획 → 실행 → 검증'**의 순서를 따르도록 설계합니다.

1.  **[Step 1: 진단 및 입력]**
    *   사용자는 '목표 설정' 페이지에서 자신의 최종 목표와 현재 상태를 입력한다. (위 A 필드 입력)
2.  **[Step 2: 로드맵 생성 (AI Core)]**
    *   시스템은 입력된 데이터를 기반으로 **3단계 맞춤형 마스터 로드맵(Phase-based Roadmap)**을 자동 생성한다.
        *   *Phase 1:* 기초 다지기 (Foundation)
        *   *Phase 2:* 실행 및 적용 (Execution & Application)
        *   *Phase 3:* 심화 및 통합 (Mastery & Integration)
    *   각 단계는 구체적인 행동 목록(Action Items), 예상 소요 시간, 필요한 리소스(비용/정보)를 포함한다.
3.  **[Step 3: 상세 Q&A 세션]**
    *   사용자는 생성된 로드맵 중 특정 단계에서 막히는 지점(예: Phase 2의 '법적 서류 준비' 관련 질문)을 AI에게 즉시 던진다.
    *   AI는 **맥락 기반**으로, 사용자가 입력한 초기 데이터와 현재 로드맵의 맥락을 모두 고려하여 구체적인 답변과 추가 리소스를 제공한다. (이것이 AI Q&A의 핵심 가치).
4.  **[Step 4: 실행 및 피드백]**
    *   사용자는 생성된 액션 아이템을 자신의 일정에 반영하고, 완료 후 결과를 다시 시스템에 입력하여 다음 단계 로드맵 수정에 반영한다. (반복 학습 루프 구축)

## 2. 티어별 가치 제안 및 가격 전략

수익화의 핵심은 **'노력 대비 결과(Effort vs. Outcome)'**를 명확히 구분하는 것입니다. Basic은 '정보 제공', Pro는 '실행 지원 및 개인화된 코칭'에 중점을 둡니다.

| 티어 (Tier) | 이름 (Proposed Name) | 주요 기능 범위 | 핵심 가치 제안 (Value Proposition) | 예상 가격대 (Monthly) |
| :--- | :--- | :--- | :--- | :--- |
| **Basic** | **Explorer** | 제한된 정보 접근, 일반적인 FAQ 답변 (비개인화), 기본 템플릿 제공. | "한국 생활에 대한 기초 정보를 빠르게 얻고 시작할 수 있는 입문 가이드." | $9 - $15 |
| **Pro** | **Navigator** | **맞춤형 로드맵 생성**, 심층 AI Q&A (데이터 기반), 실행 계획의 자동 업데이트, 개인화된 리소스 추천. | "**당신의 목표에 최적화된, 실패 확률이 낮은 맞춤형 실행 경로를 제공하여 시간과 비용을 절약.**" | $29 - $49 |
| **Premium** | **Mastermind** | Pro 기능 전체 + 1:1 심층 컨설팅 세션 (월 1회), 커스텀 데이터 연동(외부 문서 업로드), 실시간 코칭. | "**AI를 개인 비서로 활용하여, 전문 컨설턴트 수준의 깊이와 실행력을 확보.**" | $79 - $149+ |

### 전략적 근거 (Rationale)

1.  **Explorer (Basic):** 낮은 진입 장벽을 제공하여 시장에 최대한 많은 사용자를 유입시키고, 잠재 고객의 Pain Point를 확인하게 합니다.
2.  **Navigator (Pro - 핵심 목표):** 가장 높은 지불 의사를 가진 사용자(실제로 문제를 해결하고 싶어 하는 사용자)에게 **자동화된 실행 계획**이라는 강력한 가치를 제공합니다. 이는 단순 정보 검색이 아닌, **'해결책을 위한 코칭 엔진'**으로 포지셔닝합니다.
3.  **Mastermind (Premium):** 시간과 비용을 절약하는 것을 넘어, '전문가의 개입'에 대한 프리미엄 가격을 책정하여 최상위 사용자의 니즈를 충족시킵니다.
